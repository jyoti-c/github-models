# Free OpenAI API via GitHub Models Demo

This repository contains the code for the JKC Tutorials video: **Stop Paying for OpenAI APIs (Use This Instead)**. 

It demonstrates how to drop a free, unified inference endpoint into a standard OpenAI Python script to prototype with models like `gpt-4o`, `DeepSeek-R1`, and `Llama-3` for free.
